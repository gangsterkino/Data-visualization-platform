## Install

```sh
npm i install -no-fund
npm install @vicons/ionicons5
npm install chart.js
npm install axios
```
## Running
backend
```sh
python manage.py runserver 8001
```
frontend
```sh
npm run dev
```


```sh
npm run build
```

运行可能遇到报错Permission denied（没有权限）解决方法，按以下步骤:
```sh
chmod 777 node_modules/.bin/vue-cli-service
npm run build
```
